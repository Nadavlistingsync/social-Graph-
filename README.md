# Social Graph

Obsidian-style relationship network for warm intros — not a CRM.

## Killer workflow

**Who is the best person I know who can credibly get me to this target?**

## Run the MVP

```bash
cd web
npm install
npm run dev
```

Open the app, then:

1. **Graph** — explore the seeded NYC real-estate network (zoom, filter edge types, hide weak links, highlight intro path)
2. **Person** — open any node as a note with citations, tags, private notes, backlinks
3. **Path Finder** — set target to Trump (or anyone) → see ranked paths; verdict surfaces **Ask Jay Neveloff**
