# Social Graph

**One job:** Who do I know who can get me to this person?

Not a CRM. A small relationship map for warm intros.

## Three screens

1. **Find path** — pick a target → see who to ask first
2. **Graph** — look at the network
3. **Note** — open a person; see why each link exists

## Automatic vs manual

| Automatic | Manual |
|-----------|--------|
| Public relationships + sources | Who you actually know |
| Path search | Private notes |
| Link strength from evidence | “This intro is awkward” judgment |

Public structure is automatic. Access is manual. A weak mention is never a warm intro.

## Look & feel

- **Like Obsidian:** graph, notes, clickable links, quiet dark UI
- **A little like a CRM:** tags / “I know them” only as side metadata
- **Never:** spreadsheets, pipelines, uncited AI claims, dashboard clutter
